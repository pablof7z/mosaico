# workspace (replaces project concept)

The machine+path binding a root channel may carry — formerly called 'project_roots', now 'workspace_roots'. Each daemon has its own local state.db, so workspace is inherently per-machine. The human-facing hook wrapper renders <workspace name="X"> instead of <project>.
